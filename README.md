# 📊 Customer Churn Prediction & Dashboard

## 📌 Project Overview

This project focuses on predicting customer churn using machine learning techniques and visualizing key business insights through an interactive Power BI dashboard.

The goal is to identify customers who are likely to churn and understand the factors influencing churn, enabling businesses to take proactive retention actions.


## 🎯 Objectives

* Predict whether a customer will churn or not
* Identify key factors affecting churn
* Build an interactive dashboard for business insights
* Support data-driven decision making


## 🛠️ Tools & Technologies

* Python (Pandas, NumPy)
* Matplotlib, Seaborn
* Scikit-learn (Machine Learning)
* Jupyter Notebook
* Power BI


## 📂 Dataset

* Customer churn dataset
* Includes features like:

  * Age
  * Gender
  * Contract type
  * Monthly charges
  * Tenure
  * Payment method
* Target variable: **Churn (Yes/No)**


## 🔍 Data Analysis & Preprocessing

* Data cleaning and handling missing values
* Encoding categorical variables
* Feature scaling
* Exploratory Data Analysis (EDA)
* Outlier detection


## 🤖 Machine Learning Models

* Logistic Regression
* Decision Tree
* Random Forest

### 📈 Model Evaluation

* Accuracy
* Precision
* Recall
* Confusion Matrix


## 📊 Power BI Dashboard

An interactive dashboard was created to analyze churn patterns and business metrics.

### 🔹 Key Features:

* KPI Cards (Total Customers, Churn Rate, Avg Charges, etc.)
* Churn analysis by contract type
* Churn by payment method
* Churn trends by tenure
* Monthly charges vs churn

### 📷 Dashboard Preview

![Dashboard](Customer_Churn_Dashboard.png)


## 💡 Key Insights

* Customers with **month-to-month contracts** have higher churn rates
* Higher **monthly charges** are associated with increased churn
* Customers with shorter tenure are more likely to churn
* Certain payment methods show higher churn behavior


## 📈 Business Recommendations

* Promote long-term contracts to reduce churn
* Offer discounts for high monthly charge customers
* Focus on customer retention in early tenure stage
* Improve customer experience for high-risk segments
